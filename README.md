# Interactive Web Development Assessment

## Overview

This project demonstrates the implementation of interactive front-end features using HTML, CSS, and JavaScript. The application focuses on user engagement, responsive design, and modern UI components.

## Features

### Task 1: Interactive UI Components

* Responsive user interface
* Modern card layouts
* Smooth animations and transitions
* Mobile-friendly design

### Task 2: Privileges Coverflow Carousel

* Custom coverflow slider
* Previous and Next navigation controls
* Responsive layout for desktop, tablet, and mobile devices
* Smooth slide transitions

### Task 3: Lead Generation Form

* Embedded YouTube video
* User engagement tracking
* Lead capture popup displayed after user interaction
* Form validation
* Success notification after submission

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

├── index.html

## How to Run

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Open the project folder.

3. Launch `index.html` in a browser or use a local development server such as Live Server.

## Key Functionalities

### Responsive Design

The application adapts to different screen sizes using CSS media queries.

### Carousel Navigation

The privileges section uses a coverflow-style carousel with responsive navigation arrows and slide animations.

### Lead Generation Workflow

1. User interacts with the video section.
2. A timer tracks engagement.
3. A lead capture form appears after a delay.
4. User submits details.
5. Success confirmation is displayed.
